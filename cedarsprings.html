Cedar Springs Realty's website has foundational SEO problems that are likely costing it visibility in local search results. While the site runs on HTTPS and has a valid sitemap, it's missing critical elements that Google uses to understand, rank, and display the business in search results.

The biggest issues: no blog or content strategy (just 15 pages total, many of them individual listing pages), almost no structured data (only a basic WebSite schema, missing LocalBusiness, RealEstateAgent, and FAQ markup), duplicate H1 tags, very thin internal linking, and no evidence of local SEO optimization beyond the meta description mentioning Raleigh and Durham.

There are also quality issues like misspelled URLs ("/maintenence" and "/owener-faqs"), empty alt text on images, and no lazy loading for images. The site appears built on a website builder platform (DudaMobile/Duda) rather than a CMS that gives full SEO control.

The good news: the meta description is well-written, HTTPS is properly configured with HSTS, the site loads relatively quickly, and the Open Graph tags are set up correctly for social sharing. These are solid foundations to build on.

Technical SEO
✓
HTTPS & Security Headers
Site serves over HTTPS with HTTP/2. Strict-Transport-Security header is set with max-age=31536000 and preload. X-Content-Type-Options: nosniff and X-Frame-Options: SAMEORIGIN are also present. Solid security configuration.
✓
robots.txt
robots.txt is present and correctly references the sitemap. No pages are blocked from crawling. All user agents are allowed.
User-agent: *
Sitemap: https://www.cedarspringsrealty.com/sitemap.xml
⚠
XML Sitemap Issues
Sitemap exists with 42 URLs, but has problems: (1) All pages have priority set to 1.0, which defeats the purpose of priority signals. Google essentially ignores identical priority values. (2) All lastmod dates are the same (Feb 18, 2026) except the homepage and one other page, suggesting they aren't being dynamically updated. (3) No image sitemap entries for property photos.
✓
Canonical Tag
Self-referencing canonical tag is properly set on the homepage.
<link rel="canonical" href="https://www.cedarspringsrealty.com/">
✓
Gzip Compression
Compression is enabled. The homepage compresses from 181 KB to 38 KB (79% reduction). Good performance.
✓
Page Load Time (Server Response)
Server response time (TTFB) is fast at ~168ms. The server is responsive, though full page load with rendering may vary for end users.
✗
No Lazy Loading on Images
None of the 13 images on the homepage use loading="lazy". This means all images load immediately regardless of whether they're visible, slowing initial page render for visitors, especially on mobile.
✗
Misspelled URLs in Sitemap
Two URLs in the sitemap contain spelling errors. These create a poor impression for users who see them in the browser bar and can cause confusion if the correct spellings are ever created as separate pages.
/maintenence → should be /maintenance
/owener-faqs → should be /owner-faqs
⚠
Cache Control Set to No-Cache
The homepage returns Cache-Control: no-cache, must-revalidate with an expired date. This forces browsers to re-download the page on every visit instead of caching it. Static assets should allow browser caching to improve repeat visit performance.
✓
Mobile Viewport
Viewport meta tag is properly configured with initial-scale=1 and viewport-fit=cover. Max-scale is set to 5, which allows users to zoom (good for accessibility).

On-Page SEO
⚠
Title Tag
The title tag exists and includes the brand name, but it's generic and doesn't target any local keywords that potential clients would search for.
Cedar Springs Realty | Real Estate & Management
Recommendation: "Property Management in Raleigh & Wake Forest NC | Cedar Springs Realty" would target actual search queries while keeping the brand name.
✓
Meta Description
Well-written meta description at 156 characters. Includes location keywords (Raleigh, Durham, Triangle), service keywords (real estate, property management), and a differentiator (Veteran-owned). Good job here.
Full-service real estate sales and property management in Raleigh, Durham, and the Triangle. Veteran-owned, mission-driven, and dedicated to your best interests.
✗
Duplicate H1 Tags
The homepage has two H1 tags, and both just say "CEDAR SPRINGS REALTY." Multiple H1 tags dilute heading hierarchy, and using just the company name wastes the most important on-page ranking signal. The H1 should describe what the business does and where, not just repeat the brand name.
H1 #1: CEDAR SPRINGS REALTY (desktop version)
H1 #2: CEDAR SPRINGS REALTY (mobile version)
Recommendation: Use a single H1 like "Property Management & Real Estate in the Triangle, NC" and use CSS to handle responsive display instead of duplicate elements.
⚠
Heading Hierarchy
The heading structure skips levels and uses generic text. H2s are "Welcome Home," "Newest Listings," "Testimonials," and "Contact Us." None of these contain keywords that searchers would use. The H3 tags include "Tenants" and "Owners" which are vague. H4 tags appear below but the hierarchy isn't clean.
✗
Thin Internal Linking
The homepage body content contains almost no text-based internal links. The only internal link found in the page content is to the homepage itself and the navigation menu. Good SEO practice calls for contextual links within body copy pointing to service pages, listing pages, and informational content. Navigation links alone aren't enough.
⚠
Image Alt Text
13 images on the homepage. 11 have alt text, but 2 have empty alt attributes, and several alt texts are generic rather than descriptive. "A blank, white square" appears twice (spacer images). "Quote 1," "Quote 2," and "Quote 3" for testimonial images miss an opportunity to include the reviewer's name.

Minimal On-Page Content
The homepage has very little readable text content. The main body paragraph is a single block about the company. There's no service-focused content, no area-specific content, no FAQ section, and no calls-to-action beyond "About Us" and "Contact." Google needs text content to understand what the page is about and rank it for relevant searches.
✗
No Blog or Content Hub
The sitemap shows only 15 unique pages (excluding listing detail pages). There's no blog, no resource center, no area guides, and no educational content. For a local property management company, content like "Guide to Renting in Wake Forest," "Property Owner Tax Tips in NC," or neighborhood guides would attract organic search traffic and establish authority.
⚠
Open Graph Tags
OG tags are present and correctly configured (og:title, og:description, og:image, og:url, og:type). The OG image points to the company logo. A custom social sharing image with branding and a tagline would perform better when shared on social media.

Content Quality & E-E-A-T
✗
Experience & Expertise Signals
The homepage mentions being "Veteran-owned" in the meta description but doesn't highlight this on the page itself. There's no team bios, no credentials, no years of experience callout, and no specific market expertise demonstrated. Google's E-E-A-T guidelines reward content that shows real experience and expertise. The About Us page likely has some of this, but the homepage should showcase it too.
⚠
Testimonials Section
The homepage includes a testimonials section with at least 3 quotes, which is a trust signal. The content includes a genuine 5-star review mentioning specifics about Youngsville and PM experience. This is good, but could be strengthened with schema markup (Review schema), full names, and photos.
✗
Search Intent Mismatch
Someone searching "property management Raleigh NC" wants to quickly understand services offered, pricing, coverage area, and why to choose this company. The homepage doesn't clearly answer these questions above the fold. The main content paragraph is corporate-sounding and doesn't speak directly to pain points property owners or tenants have.
✗
Content Depth
With only ~150 words of body content on the homepage, Google has very little to index. Competing property management companies in Raleigh/Durham likely have 500-1000+ words on their homepages covering services, areas served, and value propositions. The current thin content makes it very difficult to rank for any competitive terms.
✓
Spelling Note in Main Content
The banner text reads "Serving the Wake Forest, Raliegh, Durham, Chapel Hill" with "Raleigh" misspelled as "Raliegh." This is a small but noticeable credibility issue, especially for a local company whose prospects are in Raleigh. Note: the meta description spells it correctly.
📍
Local SEO
✗
LocalBusiness Schema Missing
The only structured data on the site is a basic WebSite schema with the name and URL. There's no LocalBusiness, RealEstateAgent, or Organization schema. This means Google can't easily extract the business name, address, phone number, hours, service area, or reviews from the page. This is one of the highest-impact fixes available.
Current schema:
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "Cedar Springs Realty",
  "url": "https://www.cedarspringsrealty.com/"
}
⚠
NAP Consistency (Name, Address, Phone)
Business information is present in the footer but not in a structured, crawlable format. The address (3650 Rogers Road #284, Wake Forest NC 27587), phone ((919) 998 8613), and email (carl@cedarspringsrealty.com) are visible but embedded in paragraph text rather than in structured address markup. No phone number uses a tel: link for click-to-call on mobile.
✗
No Service Area Pages
The site mentions serving Wake Forest, Raleigh, Durham, and Chapel Hill, but there are no dedicated pages for each city or area. Location-specific pages (e.g., "Property Management in Raleigh NC") are one of the most effective ways to rank in local search. Each service area should have its own page with unique content about that market.
✗
No Google Business Profile Link
There's no link to a Google Business Profile from the website, and no embedded Google Map. A GBP link and embedded map help Google verify the business location and can improve local pack rankings. If a GBP exists, it should be linked. If one doesn't exist, creating one should be a top priority.
✗
No FAQ Schema on FAQ Pages
The site has three FAQ pages (applicant-faqs, owner-faqs, resident-faqs) but the homepage doesn't include FAQ schema and there's no evidence of FAQPage structured data. FAQ schema can generate rich results in Google showing questions and answers directly in search, increasing click-through rates significantly.
✗
No Social Media Links
No links to Facebook, Instagram, LinkedIn, or any social media profiles were found on the homepage. Social profiles help establish brand identity, provide additional ranking signals, and allow Google to associate the business entity across platforms.
🎯
Prioritized Action Plan
1
Add LocalBusiness / RealEstateAgent Schema High Impact
Implement comprehensive JSON-LD structured data including business name, address, phone, service areas, hours, and aggregate rating. This directly improves how Google understands and displays the business in search results and map packs.
2
Create City/Service Area Landing Pages High Impact
Build dedicated pages for each service area: Raleigh, Durham, Wake Forest, Chapel Hill, and Youngsville. Each page should include unique content about property management in that specific market, local market insights, and service details. This is the single biggest lever for ranking in local searches.
3
Fix the H1 Tag and Heading Hierarchy High Impact
Replace the duplicate "CEDAR SPRINGS REALTY" H1s with a single, keyword-rich H1 like "Property Management & Real Estate Services in the Triangle, NC." Update H2s to include service-focused language. Use CSS for responsive styling instead of duplicate heading elements.
4
Optimize the Title Tag High Impact
Change from "Cedar Springs Realty | Real Estate & Management" to something targeting actual search queries: "Property Management in Raleigh & Wake Forest NC | Cedar Springs Realty." The title tag is the most visible ranking element in search results.
5
Add Substantial Homepage Content High Impact
Expand the homepage from ~150 words to 500-800 words. Include sections covering: services offered, areas served, why choose Cedar Springs (veteran-owned, since 2015), property types managed, and a clear call-to-action. This gives Google content to index and rank.
6
Launch a Blog / Resource Center Medium Impact
Start publishing 2-4 articles per month targeting questions property owners and tenants in the Triangle ask: rental market updates, tenant screening tips, NC landlord-tenant law guides, maintenance checklists, neighborhood spotlights. This builds topical authority and drives organic traffic over time.
7
Fix Misspelled URLs and Content Medium Impact
Correct "/maintenence" to "/maintenance" and "/owener-faqs" to "/owner-faqs" with proper 301 redirects. Fix "Raliegh" to "Raleigh" in the banner text. Small credibility issues like these erode trust with visitors.
8
Add FAQ Schema to FAQ Pages Medium Impact
Implement FAQPage structured data on the three existing FAQ pages (applicant, owner, resident). This can generate rich results in Google that show expandable Q&A directly in search results, dramatically increasing visibility and click-through rate.
9
Improve Internal Linking Medium Impact
Add contextual text links throughout the homepage and other pages that connect to service pages, FAQ pages, and (future) blog content. Internal links help Google discover and understand page relationships, and distribute ranking authority across the site.
10
Add Lazy Loading & Optimize Images Low Impact
Add loading="lazy" to below-the-fold images. Fix empty alt attributes with descriptive text. Replace generic alt text ("Quote 1") with meaningful descriptions. Ensure all property listing images have descriptive alt text for accessibility and image search visibility.
11
Add Google Map Embed & Social Links Low Impact
Embed a Google Map on the contact page and footer. Add links to social media profiles (Facebook, Instagram, LinkedIn). Create and link a Google Business Profile if one doesn't exist. These signals help Google verify the business and improve local visibility.
12
Implement Click-to-Call Phone Links Low Impact
Wrap the phone number (919) 998 8613 in a tel: link everywhere it appears. Currently it's only plain text. On mobile, this turns the phone number into a tappable call button, which improves user experience and conversion.
