Structured Data / Schema Markup

❌
No JSON-LD Schema on Homepage

The homepage has zero structured data. No LocalBusiness, Organization, or any schema markup was detected. For a local service business, this is a major missed opportunity. Google uses LocalBusiness schema to populate local pack results, knowledge panels, and rich snippets.

Critical
⚠️
FAQPage Schema Misused on City Pages

The city pages (e.g., /raleigh-property-management, /wake-forest-property-management) declare itemtype="https://schema.org/FAQPage" on the HTML tag, but this microdata approach requires properly marked up Question/Answer pairs within the page. Without verifying the inner markup, this may be incomplete or invalid schema that Google will ignore.

6 🎯 Prioritized Action Plan
1

Add LocalBusiness + Organization Schema Markup

Add JSON-LD structured data to every page with business name, address, phone, service area, geo coordinates, hours, and aggregate rating. This is the single highest-impact local SEO change available. Use the "RealEstateAgent" or "ProfessionalService" schema type.

High Impact
2

Migrate from Universal Analytics to GA4

UA stopped processing data in July 2023. Set up GA4 via the existing GTM container, configure conversions (form submissions, phone clicks, portal logins), and start collecting data immediately. Every day without GA4 is a day of lost insights.

High Impact
3

Fix Open Graph Tags (Wrong Domain)

Update all OG tags to reference propertyspecificmanagement.com instead of propspecific.com. Make og:title, og:description, og:image, and og:url unique per page. This affects how every social share and link preview looks.

High Impact
4

Make H1 Visible and Fix Homepage Content

Remove the sr-only class from the H1, update it to accurately reflect the business (drop "Greensboro" if you don't serve it), and add 500-1000 words of helpful content to the homepage covering services, process, and value proposition.

High Impact
5

Rewrite Title Tags (Under 60 Characters)

Homepage suggestion: Raleigh Property Management | Property Specific Management (57 chars). City pages: [City] Property Management | Property Specific. Stop repeating keywords. One mention each is enough.

High Impact
6

Reactivate and Rebrand the Blog

Move blog content from /blog-hidden/ to /blog/. Start publishing 2-4 property management focused articles per month: rental market reports, landlord tips, tenant screening guides, NC landlord-tenant law updates, and local investment analyses. This is the #1 way to build organic traffic long term.

High Impact
7

Clean Up Sitemap

Remove /test, /testing, /icons/demo, /sold-report, and any other junk pages from the XML sitemap. Add a Sitemap: https://www.propertyspecificmanagement.com/sitemap.xml directive to robots.txt. Fix the unencoded space in the blog URL slug.

Medium Impact
8

Standardize Brand Name Everywhere

Decide between "Property Specific Management" and "Property Specific Realty" and use it consistently across the website, OG tags, Google Business Profile, Yelp, and all directory listings. NAP inconsistency is one of the top local SEO ranking killers.

Medium Impact
9

Create Landing Pages for 5 Missing Service Cities

Build unique, content-rich landing pages for Youngsville, Fuquay Varina, Wendell, Franklinton, and Holly Springs. Each page should have 300-500+ words of unique content about that city's rental market, local ordinances, and neighborhood highlights.

Medium Impact
10

Add Security Headers

Configure HSTS, Content-Security-Policy, X-Frame-Options, and Referrer-Policy headers on the nginx server. Also hide the server version string. This improves security posture and can contribute to positive trust signals.

Medium Impact
11

Add E-E-A-T Content to About Page

Showcase Carl Driver's experience, credentials, and licenses. Add the number of doors managed (~200), years in business, NARPM membership (if applicable), NC broker license number, and specific expertise areas. Link team bios to LinkedIn profiles.

Medium Impact
12

Modernize Frontend Dependencies

Upgrade from jQuery 1.12.4 and Bootstrap 3.3.6 to modern alternatives. Remove IE conditional comments and polyfills. Convert images to WebP format. These changes will improve Core Web Vitals scores (LCP, CLS, FID) and reduce page weight.

13

Improve Meta Descriptions with CTAs

Write unique meta descriptions for each page that include a call to action. Example homepage: "Property Specific Management provides full-service property management in Raleigh and Wake County. Managing 200+ doors with customized plans. Get a free rental analysis today." (155 chars)

Add Review/Testimonial Schema
